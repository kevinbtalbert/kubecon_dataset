General Information
Thank you for your support of Cloud Native Security Day EU 2020, taking place on March 30th at the RAI Amsterdam.
Please click through the tabs on this page to access information on exhibiting at the conference.
Registration
All sponsors should register here using the sponsor code that was emailed to you. *Please note: You must pay the registration fee for Kubecon + Cloud Native Con. This discount code only waives the Cloud Native Security Day registration fee.



Hotel Information
Please visit the Kubecon + Cloud Native Con venue and travel page for hotel information.



Due Dates
February 28th: Logo in .svg format due.February 28th: Speaker information due. (Diamond only)March 6th: Register attendees by this date.March 6th: Pre-conference email due. (Diamond only)March 13th: 150-word post event email message due. (Diamond only)



Speaking Opportunity 
Diamond sponsors receive a 5-minute keynote speaking opportunity during Cloud Native Security Day.
Please provide the required details below no later than Friday, February 28th, so it can be added to the schedule. Once the details are received, the program committee will review the content prior to it being posted. As a reminder, we encourage you to use this time to benefit the community.
-Track Session Title
-Track Session Abstract (to be included on schedule)
-Speaker Name
-Speaker Title
-Speaker Bio
-Speaker Email
